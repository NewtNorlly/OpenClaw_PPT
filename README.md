# 01. 什么是 OpenClaw 🧠
OpenClaw 是一个 AI Agent 平台，可以：

- 🔗 **连接 20+ 消息渠道**（WhatsApp / Telegram / 飞书 / Discord 等）  
- ⚙️ **自动执行任务**（邮件处理 / 浏览器操作 / 工具调用）  

---

## ⚖️ 核心对比

| 维度        | ChatGPT 🤖           | OpenClaw 🦞              |
|-------------|---------------------|--------------------------|
| 交互模式    | 你问它答             | 自动执行任务             |
| 运行环境    | Web / App           | 自托管服务器             |
| 数据控制    | OpenAI 云端         | 完全本地可控             |
| 开源        | ❌                  | ✅ MIT License           |

---

# 02. 技术架构 🏗️

采用 **Gateway - Node - Channel 三层架构**

### 🔧 核心组件
- 🧭 **Gateway**：控制中心（Session + 调度）  
- 💻 **Node**：执行节点（本地操作）  
- 🌐 **Channel**：消息接入层  

### 🔐 安全设计
- 🚫 默认不开放外网端口  
- 🌍 使用 Tailscale 实现安全远程访问

# 03. 部署方案 🚀
