## 01. OpenClaw 是什么
OpenClaw 是一个开源、自托管的AI Agent系统，它能将AI从「聊天工具」变成「能自主执行任务的数字员工」。

- **功能**: 连接20+消息渠道（WhatsApp、Telegram、飞书、钉钉、Discord等）
- **工作模式**: 自主执行任务，处理邮件、操作浏览器、调用各种工具

### 核心对比
| 维度      | ChatGPT               | OpenClaw                |
|-----------|-----------------------|-------------------------|
| 交互模式  | 你问它答              | 自主执行任务            |
| 运行环境  | 网页/App              | 自托管服务器，接入20+平台|
| 数据控制  | 数据在OpenAI          | 完全本地，你拥有所有数据|
| 开源      | 否                    | 是，MIT License         |

## 02. 技术架构
OpenClaw 使用 Gateway-Node-Channel 三层架构，以 WebSocket 为通信总线，将控制平面、设备执行与消息渠道解耦。

### 组件介绍
- **Gateway**: 中央控制平面，负责管理 Session 和调度 Agent
- **Node**: 设备端执行节点，执行本地操作
- **Channel**: 消息渠道接入层，支持 WhatsApp、Telegram、飞书等

### 安全设计
- 默认不开放任何外网端口
- 使用 Tailscale Serve/Funnel 实现远程访问

### 部署方式

- **云部署**: 提供一键部署方案，支持阿里云、腾讯云等国内云厂商
